Programming Languages
My programming language of choice is Java, it is what I've had the most experience with. I'm comfortable with HTML, CSS, and Javascript for web development purposes. I have also dabbled with Python and Lua, having an understanding of the syntax.
Software
I use a variety of software in order when working on personal projects. For Image Editing I stick to Adobe Photoshop and have years of experience with it. For Video Editing my preference is to use Sony Vegas and Adobe After Effects. When it comes to Audio Editing, I rely on the use of Audacity. When programming I often will use an IDE or just a basic text editor like NotePad++ or SublimeText. For Java my favorite IDE is IntelliJ by JetBrains.
